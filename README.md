# Interactive Landing Page Navigation

##  Project Overview

This project is a simple interactive landing page featuring a fixed navigation menu that changes style when scrolling and hovering over menu items.

##  Features

* Fixed navigation bar
* Color change on scroll
* Hover effects on menu items
* Smooth section navigation
* Responsive layout

##  Technologies Used

* HTML
* CSS
* JavaScript

##  Project Structure

landing-page/
│── index.html
│── style.css
│── script.js

##  How to Run

1. Download or clone the repository
2. Open the folder in VS Code
3. Run using Live Server
4. Open in browser:

  -- http://127.0.0.1:5500/index.html
   
##  Purpose

This project is ideal to understand:

* DOM manipulation
* Event handling
* Responsive navigation design
